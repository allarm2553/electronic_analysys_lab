# คู่มือระบบจำลองวงจรเรียงกระแสครึ่งคลื่น (Half-Wave Rectifier Simulation E-Lab App)

โปรเจกต์นี้คือใบงานการทดลองและแอปพลิเคชันจำลองการทำงานของ **วงจรเรียงกระแสแบบครึ่งคลื่น (Half-Wave Rectifier)** ที่ออกแบบมาให้รองรับการทำงานในแบบโต้ตอบ (Interactive) ด้วยเครื่องมือวัดเสมือนจริง ได้แก่ **ดิจิทัลมัลติมิเตอร์ (DMM)** และ **ออสซิลโลสโคปดิจิทัล (Digital Oscilloscope)** และสามารถเกรดคะแนนใบงานพร้อมบันทึกผลเข้าระบบ **Google Sheets** โดยอัตโนมัติ

---

## 📂 รายละเอียดไฟล์ในโฟลเดอร์นี้
- `Code.js` - สคริปต์เซิร์ฟเวอร์ฝั่ง Google Apps Script รองรับการจำลองใบงาน, เก็บคะแนนประเมินอัตโนมัติ และบันทึกประวัติการส่งลงแผ่นชีต
- `index.html` - ไฟล์เอกสาร HTML/CSS/JS รวม (Compiled Build) สำหรับนำไปจัดทำ Web App บนหน้า Google Apps Script
- `assemble_app.py` - สคริปต์คอมไพล์สำหรับรวบรวมโค้ดและส่วนการออกแบบสไตล์ชีตจากตัวจำลองในโฟลเดอร์ภายนอก

---

## 🔬 สเปกการวัดและการคำนวณในแบบจำลอง
- **Input Voltage:** AC 12 Vrms, 50 Hz (ยอดแรงดันสัญญาณอินพุตสูงสุด V_peak ≈ 16.97 V)
- **ไดโอด (Diode):** 1N4007 (ค่าเฉลี่ยแรงดันตกคร่อมรอยต่อ V_D = 0.7 V)
- **ตัวต้านทานโหลด (R_L):** 1 kΩ
- **ค่าเฉลี่ยแรงดันขาออก (Output V_dc):** (V_peak - V_D) / π ≈ 16.27 / π ≈ 5.18 V
- **ค่ากระแสไฟฟ้าตรงโหลด (Output I_dc):** V_dc / R_L ≈ 5.18 mA

---

## 🚀 วิธีการทดสอบคอมไพล์โครงการ (Local Compilation)
หากมีการปรับแต่งรูปแบบ CSS หรือโค้ดจำลองในไดเรกทอรี `lab-simulators/rectifier-lab-simulator` คุณสามารถคอมไพล์โค้ดใหม่เข้าสู่ไฟล์ `index.html` ของโฟลเดอร์นี้โดยรันคำสั่ง:

```bash
python assemble_app.py
```

เมื่อเสร็จสิ้นจะมีหน้าข้อความเตือน `GAS index.html successfully compiled!` และไฟล์ `index.html` จะอัปเดตโดยตรง

---

## 📊 วิธีการติดตั้งและใช้งานร่วมกับ Google Sheets
1. สร้าง Google Sheet แผ่นใหม่ขึ้นมาในไดรฟ์ของท่าน
2. ไปที่เมนู **ส่วนขยาย (Extensions)** -> **Apps Script**
3. คัดลอกและนำเนื้อหาไฟล์ `Code.js` ไปวางแทนที่โค้ดทั้งหมดในไฟล์ `Code.gs` ของหน้าต่างสคริปต์
4. สร้างไฟล์ HTML ใหม่ในหน้านั้น ตั้งชื่อไฟล์ว่า `index` และคัดลอกเนื้อหาทั้งหมดในไฟล์ `index.html` ของโฟลเดอร์นี้ไปวางแทนที่
5. กดปุ่มบันทึกโครงการ 💾
6. จัดการ Deploy ใช้งานจริง (New deployment):
   - ตั้งค่าประเภทเป็น **Web App (เว็บแอป)**
   - **Execute as (รันในนาม):** ให้เลือกเป็น **`User accessing the web app`** (ผู้ใช้ที่เข้าถึงแอปเว็บ)
   - **Who has access (ผู้มีสิทธิ์เข้าถึง):** เลือก **`Anyone`** (ทุกคน) หรือจำกัดเฉพาะโดเมนของโรงเรียน/องค์กร
7. คัดลอกลิงก์ที่ได้จากการดีพลอยไปเปิดใช้งานสำหรับบันทึกค่าวัดและประเมินคะแนนผู้เรียนทันที!
