# 📊 คู่มือการติดตั้งโปรเจกต์ Google Sheets E-Lab Web App (ใบงานที่ 2: ซีเนอร์ไดโอด)

คู่มือนี้สำหรับอาจารย์ผู้สอนเพื่อเปิดใช้หน้าแล็บออนไลน์จำลอง **Lab 2: ซีเนอร์ไดโอด (Zener Diode Characteristics)** พร้อมระบบ **ตรวจให้คะแนนอัตโนมัติ (Auto-Grading)** และบันทึกลง Google Sheets ของอาจารย์โดยตรง

---

## 🛠️ ขั้นตอนที่ 1: เตรียม Google Sheet
1. เปิด [Google Sheets](https://sheets.google.com/) แล้วสร้างชีทใหม่ (Blank Spreadsheet)
2. ตั้งชื่อชีทตามต้องการ เช่น `บันทึกคะแนนปฏิบัติการซีเนอร์ไดโอด`
3. ในแผ่นชีทแรก ให้เปลี่ยนชื่อชีท (Rename) จากเดิม "Sheet1" ให้เป็น **`Submissions`** (ตัวสะกด S ใหญ่ และมี s ด้านหลัง) เพื่อรองรับการเขียนข้อมูลคะแนนของระบบ

---

## 💻 ขั้นตอนที่ 2: คัดลอกโค้ดไปยัง Google Apps Script
1. ที่แถบเมนูด้านบนของ Google Sheet ให้คลิกที่ **Extensions (ส่วนขยาย)** -> **Apps Script**
2. หน้าโปรเจกต์สคริปต์จะถูกเปิดขึ้นมา ให้เปลี่ยนชื่อโปรเจกต์มุมบนซ้ายตามต้องการ เช่น `Zener Lab 2 Web App`
3. ในส่วนของหน้าต่างไฟล์ทางด้านซ้าย:
   - ที่ไฟล์ **`Code.gs`** (หลังบ้าน): ให้ลบโค้ดเดิมออกทั้งหมด แล้วนำโค้ดในไฟล์ [Code.js](file:///C:/Users/terd2/OneDrive/Desktop/diode-lab-simulator/elab-zener-sheets-app/Code.js) ไปวางแทนที่แล้วกดบันทึก 💾
   - กดปุ่ม **`+` (Add a file)** ข้างคำว่า Files -> เลือก **`HTML`** -> ตั้งชื่อไฟล์ว่า **`index`** (ไม่ต้องใส่ดอต html)
   - ที่ไฟล์ **`index.html`** (หน้าบ้าน): ให้ลบโค้ดเริ่มต้นออกให้หมด แล้วนำโค้ดทั้งหมดในไฟล์ [index.html](file:///C:/Users/terd2/OneDrive/Desktop/diode-lab-simulator/elab-zener-sheets-app/index.html) ไปวางแทนที่แล้วกดบันทึก 💾

## 🚀 ขั้นตอนที่ 3: เริ่มเปิดใช้งานเว็บแอป (Deployment)
1. ที่มุมขวาบนของหน้า Apps Script ให้คลิกปุ่ม **`Deploy (การใช้งานได้จริง)`** -> เลือก **`New deployment (การทำงานร่วมกับการตั้งค่าใหม่)`**
2. คลิกปุ่มฟันเฟืองข้าง "Select type" -> เลือก **`Web app (เว็บแอป)`**
3. ตั้งค่ารายละเอียดดังนี้:
   - **Description**: ตั้งคำอธิบายตามใจชอบ เช่น `V1.0 Launch`
   - **Execute as (รันในสิทธิ์ของ)**: เลือก **`Me (อีเมลผู้สอน)`** (เพื่อให้หน้าเว็บเขียนข้อมูลลงชีทของคุณได้แทนสิทธิ์ของนักเรียน)
   - **Who has access (ผู้มีสิทธิ์เข้าถึง)**: เลือก **`Anyone (ทุกคน)`** หรือ **`Anyone with Google account`** เพื่อให้นักศึกษาสามารถเข้าทำแล็บได้โดยไม่ต้องขอสิทธิ์เพิ่ม
4. คลิกปุ่ม **`Deploy`**
5. หากระบบขอสิทธิ์เข้าถึงข้อมูลของชีท (Authorize Access):
   - คลิก **`Authorize Access`** -> เลือกบัญชี Google ของคุณ -> คลิก **`Advanced (ขั้นสูง)`** -> เลือก **`Go to ... (unsafe)`** -> คลิก **`Allow (อนุญาต)`**
6. เมื่อ Deploy สำเร็จ ระบบจะแสดงลิงก์ **`Web app URL`**
7. ให้ทำการ **คัดลอกลิงก์ Web app URL** นี้ส่งต่อให้นักศึกษาเพื่อเข้าทำใบงานการทดลองและรับคะแนนตรวจสดได้ทันทีครับ!

## 📊 รายการข้อมูลที่จะถูกบันทึกใน Sheet อัตโนมัติ:
เมื่อนักศึกษากดส่งใบงาน ระบบจะตรวจคะแนน (เต็ม 10 คะแนน) และเขียนข้อมูลลงชีท `Submissions` โดยจัดทำเป็นคอลัมน์ให้อัตโนมัติ:
1. **Timestamp**: วันและเวลาที่กดส่ง
2. **Student Name**: ชื่อ-นามสกุลนักศึกษา
3. **Student ID**: รหัสนักศึกษา
4. **Group**: กลุ่มเรียน
5. **Lab Date**: วันที่ทำแล็บ
6. **Zener Condition**: สภาพซีเนอร์ไดโอดที่ทดสอบจริง (ดี / ขาด / ลัดวงจร)
7. **Auto Score**: คะแนนจริงที่ได้ (คะแนนเต็ม 10 คะแนน)
8. **Evaluation**: ระดับประเมินผล (เช่น ผ่านเกณฑ์ดีมาก)
9. **Feedback Summary**: รายละเอียดการหักคะแนนแต่ละข้อแบบละเอียด
10. **Q1 Answer**: คำตอบคำถามท้ายการทดลอง ข้อ 1
11. **Q2 Answer**: คำตอบคำถามท้ายการทดลอง ข้อ 2
12. **Q3 Answer**: คำตอบคำถามท้ายการทดลอง ข้อ 3
13. **Conclusion**: สรุปผลการทดลองเขียนรายงาน
