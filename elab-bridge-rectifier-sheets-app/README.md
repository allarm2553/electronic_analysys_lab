# คู่มือระบบจำลองวงจรเรียงกระแสเต็มคลื่นแบบบริดจ์ (Full-Wave Bridge Rectifier E-Lab App)

โปรเจกต์นี้คือใบงานการทดลองและแอปพลิเคชันจำลองการทำงานของ **วงจรเรียงกระแสเต็มคลื่นแบบบริดจ์ (Full-Wave Bridge Rectifier)** ที่รองรับการวัดค่าวัดด้วยเครื่องมือวัดเสมือนจริง ได้แก่ **ดิจิทัลมัลติมิเตอร์ (DMM)** และ **ออสซิลโลสโคปดิจิทัล (Digital Oscilloscope)** และสามารถประเมินคะแนนใบงานพร้อมบันทึกผลเข้าระบบ **Google Sheets** โดยอัตโนมัติร่วมกับอีเมลผู้เรียน

---

## 📂 รายละเอียดไฟล์ในโฟลเดอร์นี้
- `Code.js` - สคริปต์ตรวจคะแนนอัตโนมัติ และบันทึกประวัติการส่งลง Google Sheet
- `index.html` - ไฟล์แอปพลิเคชัน HTML/CSS/JS รวม (Compiled Build) สำหรับไปติดตั้งบน Google Apps Script Web App
- `assemble_app.py` - สคริปต์คอมไพล์เพื่อรวบรวมโค้ดและการออกแบบจากโฟลเดอร์ต้นทางจำลอง

---

## 🔬 สเปกการวัดและการคำนวณในแบบจำลอง
- **Input Voltage:** AC 12 Vrms, 50 Hz (V_peak ≈ 16.97 V)
- **การนำกระแสของไดโอดบริดจ์:** นำกระแสครั้งละ 2 ไดโอด ทำให้นักศึกษาวัดแรงดันตกคร่อมรวมได้ $V_{rect\_drop} = 2 \times 0.7 = 1.4\text{ V}$
- **ตัวต้านทานโหลด (R_L):** 1 kΩ
- **ค่าเฉลี่ยแรงดันเอาต์พุต (Output V_dc):** $2 \times (V_{peak} - 1.4) / \pi \approx 31.14 / \pi \approx 9.91\text{ V DC}$
- **ค่ากระแสไฟฟ้าตรงโหลด (Output I_dc):** V_dc / R_L ≈ 9.91 mA
- **ความถี่ริปเปิลเอาต์พุต (Ripple Frequency):** เพิ่มขึ้นเป็น 2 เท่าเนื่องจากทำงานเต็มคลื่น = $100\text{ Hz}$ (คาบเวลา $T_r = 10\text{ ms}$)

## 🚀 วิธีการทดสอบคอมไพล์โครงการ (Local Compilation)
เมื่อมีการปรับแก้โค้ดในตัวจำลองของโฟลเดอร์ต้นทาง ให้รันสคริปต์นี้เพื่ออัปเดตไฟล์ `index.html`:

```bash
python assemble_app.py
```

เมื่อขึ้นความสำเร็จ `GAS index.html successfully compiled!` สามารถนำไฟล์ `index.html` ไปอัปโหลดใช้งานได้ทันที

---

## 📊 วิธีการติดตั้งบน Google Sheets
1. เปิด Google Sheet แผ่นใหม่ขึ้นมา
2. เข้าไปที่ **ส่วนขยาย (Extensions)** -> **Apps Script**
3. คัดลอกและนำเนื้อหาไฟล์ `Code.js` ไปวางแทนที่โค้ดทั้งหมดในไฟล์ `Code.gs`
4. สร้างไฟล์ HTML ใหม่ ตั้งชื่อว่า `index` แล้วคัดลอกเนื้อหาทั้งหมดในไฟล์ `index.html` ไปวาง
5. กดบันทึกและจัดการ Deploy (New deployment):
   - ชนิดการ Deploy: **Web App (เว็บแอป)**
   - **Execute as (รันในนาม):** ให้เลือกเป็น **`User accessing the web app`** (ผู้ใช้ที่เข้าถึงแอปเว็บ)
   - **Who has access (ผู้มีสิทธิ์เข้าถึง):** เลือก **`Anyone`** (ทุกคน) หรือเฉพาะสถาบัน
6. นำลิงก์ Web App ไปแจกจ่ายให้นักศึกษาทำการทดลองและเกรดผลเข้าระบบชีตอย่างสมบูรณ์แบบได้ทันที!
