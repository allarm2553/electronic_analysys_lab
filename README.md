# 📊 คลังปฏิบัติการจำลองอิเล็กทรอนิกส์และระบบตรวจคะแนนอัตโนมัติ (E-Lab Sheet Apps)

ยินดีต้อนรับสู่ระบบ **E-Lab Sheet Apps** คลังห้องปฏิบัติการจำลองและการบันทึกคะแนนสะสมอัตโนมัติ (Auto-Grading) สำหรับวิชา **วิเคราะห์วงจรอิเล็กทรอนิกส์ (Electronic Circuit Analysis)** พัฒนาขึ้นในรูปแบบเว็บแอปพลิเคชันแบบตอบสนอง (Interactive Web App) เพื่อใช้ร่วมกับ Google Sheets ของผู้สอนโดยตรง

---

## 🗺️ โครงสร้างของโปรเจกต์ (Project Structure)

โปรเจกต์นี้ประกอบด้วยทางเข้าหลัก (Portal Dashboard) และใบงานย่อยจำลองจำนวน 5 ชุด:

```text
elab-sheet-apps/
├── index.html                       # หน้าหลักพอร์ทัลแผงควบคุม (Lab Portal Dashboard)
├── package.json                     # ไฟล์กำหนดการรันจำลอง Local Server (Vite)
├── .gitignore                       # ป้องกันการอัปโหลดโฟลเดอร์ที่ไม่จำเป็นขึ้น Git
├── elab-diode-sheets-app/           # [แล็บ 1] ใบงานหาลักษณะสมบัติของไดโอด (Diode Characteristics)
├── elab-zener-sheets-app/           # [แล็บ 2] ใบงานรักษาแรงดันด้วยซีเนอร์ไดโอด (Zener Regulator)
├── elab-bjt-bias-sheets-app/        # [แล็บ 3] ใบงานวงจร Fixed Bias BJT (ทรานซิสเตอร์ BC108 + แสดงกราฟจำลอง)
├── elab-transistor-sheets-app/      # [แล็บ 4] ใบงานพล็อตเส้นกราฟ BJT Output Characteristics
└── elab-mosfet-pinout-sheets-app/   # [แล็บ 5] ใบงานการจัดตำแหน่งขาและจำลอง MOSFET
```

ทุกโฟลเดอร์ของใบงานการทดลองย่อยจะประกอบด้วย:
* **`index.html`** - หน้าอินเทอร์เฟซหน้าบ้านและวงจรจำลอง (Client-side HTML/JS/CSS)
* **`Code.js`** - โค้ดควบคุมหลังบ้านสำหรับติดตั้งใน Google Apps Script (Backend)
* **`assemble_app.py`** - สคริปต์ Python สำหรับประกอบร่างและรวมโค้ดให้สะดวกต่อการใช้งาน
* **`README.md`** - คู่มือขั้นตอนการทำสำหรับนักศึกษาและผู้สอนเฉพาะใบงานนั้น ๆ

## 💻 วิธีการเปิดใช้งานบนเครื่องจำลอง (Local Development Server)

หากต้องการเปิดใช้หน้าแล็บและห้องทดลองจำลองบนเครื่องคอมพิวเตอร์ของคุณเองโดยไม่ต้องผ่านระบบคลาวด์ สามารถเปิดรัน Local Server ได้ตามขั้นตอนดังนี้:

### 1. การจัดเตรียมโปรแกรม (ครั้งแรกเท่านั้น)
* ติดตั้งโปรแกรม [Node.js](https://nodejs.org/) (รุ่น LTS)
* เปิดโปรแกรม Terminal หรือ Command Prompt ในเครื่องของคุณ และระบุตำแหน่งโฟลเดอร์มายังโปรเจกต์นี้:
  ```bash
  cd path/to/elab-sheet-apps
  ```

### 2. ติดตั้ง Dependencies
* รันคำสั่งดาวน์โหลดโมดูลเซิร์ฟเวอร์ (Vite):
  ```bash
  npm install
  ```

### 3. เปิดรันเซิร์ฟเวอร์จำลอง
* เริ่มเปิดเซิร์ฟเวอร์ด้วยคำสั่ง:
  ```bash
  npm run dev
  ```
* เมื่อเซิร์ฟเวอร์พร้อมใช้งาน จะปรากฏลิงก์จำลอง ให้คัดลอกไปวางบนเว็บเบราว์เซอร์เพื่อเริ่มใช้งาน:
  * ➜ **[http://localhost:5173/](http://localhost:5173/)**
  *(คุณจะพบกับหน้า Dashboard สไตล์ Dark Mode สวยงามที่สามารถคลิกเลือกเข้าแล็บต่าง ๆ ได้ทันที)*

## 🚀 วิธีการติดตั้งเป็นเว็บแอปจริงบน Google Sheets (สำหรับผู้สอน)

เพื่อเปิดใช้หน้าแล็บออนไลน์จำลองและระบบตรวจคะแนนบันทึกลง Google Sheets ของอาจารย์โดยตรง ให้อาจารย์ผู้สอนดำเนินการตามนี้ใน**ทุก ๆ ใบงานย่อย**ที่ต้องการเปิดให้เรียน:

### ขั้นตอนที่ 1: เตรียม Google Sheet
1. สร้าง **Google Sheet** ใหม่ใน Google Drive ของท่าน
2. เปลี่ยนชื่อแผ่นชีทแรกสุดจากเดิม "Sheet1" ให้เป็น **`Submissions`** (ตัวพิมพ์ใหญ่ S และลงท้ายด้วย s) เพื่อรองรับการเขียนข้อมูลคะแนน

### ขั้นตอนที่ 2: ตั้งค่า Google Apps Script
1. ที่แถบเมนูด้านบนของ Google Sheet ให้คลิกที่ **Extensions (ส่วนขยาย)** -> **Apps Script**
2. คัดลอกและบันทึกโค้ดหลังบ้าน:
   * เปิดไฟล์ **`Code.js`** ในโฟลเดอร์แล็บที่ต้องการ คัดลอกโค้ดทั้งหมดไปวางแทนที่โค้ดเดิมในไฟล์ **`Code.gs`** ในหน้าต่างสคริปต์
3. คัดลอกและบันทึกโค้ดหน้าบ้าน:
   * กดปุ่ม **`+` (Add a file)** ที่แถบด้านซ้าย -> เลือก **`HTML`** -> ตั้งชื่อไฟล์ว่า **`index`** (ไม่ต้องพิมพ์ .html)
   * นำโค้ดทั้งหมดในไฟล์ **`index.html`** ของโฟลเดอร์แล็บนั้น ๆ ไปวางแทนที่แล้วกดบันทึก 💾

### ขั้นตอนที่ 3: เปิดใช้งานเว็บแอป (Deployment)
1. คลิกปุ่ม **`Deploy (การใช้งานได้จริง)`** มุมขวาบน -> เลือก **`New deployment`**
2. คลิกรูปฟันเฟืองข้าง "Select type" -> เลือก **`Web app (เว็บแอป)`**
3. ตั้งค่ารายละเอียดการใช้งานดังนี้:
   * **Description**: ใส่คำอธิบายเวอร์ชัน เช่น `V1.0 Launch`
   * **Execute as (รันในสิทธิ์ของ)**: เลือก **`Me (อีเมลผู้สอน)`**
   * **Who has access (ผู้มีสิทธิ์เข้าถึง)**: เลือก **`Anyone (ทุกคน)`**
4. คลิกปุ่ม **`Deploy`** และยืนยันอนุญาตสิทธิ์เข้าถึงข้อมูลของชีท (Authorize Access)
5. ระบบจะสร้าง **Web app URL** ขึ้นมาให้อาจารย์ คัดลอกลิงก์ส่งต่อให้นักศึกษาเข้าทดลองจำลองได้ทันที

## 🌟 ฟีเจอร์เด่นในระบบตรวจคะแนนอัตโนมัติ (Auto-Grading Details)
* **Real-time BJT Curves Simulation:** ใบงาน Fixed Bias (BC108) มาพร้อมระบบจำลองกราฟสมบัติ BJT (Load Line & Q-point) เลื่อนขยับสไลเดอร์และพล็อตแบบเรียลไทม์เพื่อเพิ่มความเข้าใจในการวิเคราะห์เชิงทัศน์
* **Auto-Evaluation:** ตรวจคะแนนแบบจำลองคณิตศาสตร์ในโค้ดฝั่ง Server-side ทันทีที่นักศึกษากดส่งใบงาน (ตรวจสอบค่ากระแส ค่าแรงดันตามเงื่อนไขวงจรดี/ชำรุด และระบุตำแหน่งขั้วขาอุปกรณ์)
* **DB Logging:** บันทึก Timestamp, ชื่อ-นามสกุล, รหัสนักศึกษา, สรุปผลการทดลอง และคะแนนจริงลงสเปรดชีตของผู้สอนโดยละเอียด
